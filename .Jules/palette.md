## 2025-05-15 - [Accessible Action Buttons]
**Learning:** Icon-only buttons in UTable actions require both `aria-label` for screen readers and `UTooltip` for visual guidance to maintain high UX standards in the SNMS Portal.
**Action:** Always wrap action buttons in `UTooltip` and ensure `aria-label` is present in Polish.

## 2025-05-15 - [Form Contextual Guidance]
**Learning:** Using "np." (na przykład) prefix for placeholders in Polish interfaces provides clear, realistic examples for technical fields like MAC and IP, reducing data entry errors.
**Action:** Implement "np. [example]" placeholders for all technical and identifier input fields.
